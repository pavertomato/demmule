# how to build

1) Clone
```
git clone https://github.com/pavertomato/demmule demmule
cd demmule
git clone https://github.com/cocos2d/cocos2d-x cocos2d
```

2) Setup cocos:
```
cd cocos2d
python2 setup.py
```

3) In folder `demmule` run command using your platform key (see cocos docs):
```
cocos run -p linux
```
```
cocos run -p win32
```
```
cocos run -p mac
```
